Creating Hyper-Links

  • Hyperlink provides the ability to link from one web page to another web page or resource at your disposal
  • only anchor(<a>) element have permission to wrap any level of the element even block level
  • Relative path
    • Mostly when you have the resource and making your own web page
  • Absolute path
    • When we want to link to another web page you need an absolute path
  • Internal path
    • the link created to browse through the page using "id"
