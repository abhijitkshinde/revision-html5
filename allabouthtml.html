Request and Responce cycle
When you type a URL you make a requets and in responce the browser shows the correct or apporopriate file.
IP address determined by browser creates an http request, there is a lot of information in this request.
The srever returns the file as responce according to the requst you made
If server can't complete your request it will show you an error.
