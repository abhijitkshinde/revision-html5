The ETA

These there made the most of HTML

  • Element:
    • Designators that defines the structure and content of an object within the page
    • Example:
      • <h1>is used to give the headline
      • <p> paragraph
      • <a> for links
  • Tags:
    • Name of the element between <> is a tag
    • Example:
      • h1
      • p
      • a
  • Attributes:
    • Provides the additional information on the element
    • Example:
      • id
      • class
      • href
